## üöÄ Setup Instructions

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for best experience)

### Installation

1. **Clone or download the project**
   - Simply download the project zip file and extract

2. **Open the project**
   - Option 1: Open `index.html` directly in your browser
   - Option 2: Use a local web server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```

3. **Access the website**
   - If opened directly: File path in browser
   - If using a server: Navigate to `http://localhost:8000`

### No Build Process Required

This project uses vanilla HTML, CSS, and JavaScript with CDN-hosted dependencies, so no build tools or package managers are required. Simply open the files in a browser or serve them through a local web server.
